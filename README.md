## Descrição

Assembler para a linguagem Assembly Hack (Nand2Tetris, Projeto 06). Traduz arquivos
`.asm` em código de máquina binário de 16 bits (`.hack`).

## Build

```bash
cargo build --release
```

## Execução

```bash
cargo run --release -- test_files/add.asm
# gera test_files/add.hack
```

## Testes

```bash

```
